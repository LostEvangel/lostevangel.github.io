# Hexo 博客配置

- 博客标题：石头门的选择
- 作者：Keinz
- Hexo 版本：8.1.2
- 主题：hexo-theme-next
- GitHub Pages 部署分支：master
- Hexo 源码分支：hexo

## 工作流

- 生成文章：`hexo generate` (或 `npx hexo generate`)
- 本地预览：`hexo server`
- 部署：`git add .` → `git commit` → `git push origin hexo`（源码），然后 `hexo deploy` 或手动切 master 部署

## 文章规范

- 分类：Java、数据库、算法、网络、操作系统、Linux、工具、生活、Python
- 常用标签：面试、源码分析、后端开发、并发编程、JVM、Spring、设计模式、中间件
- 日期格式：`YYYY-MM-DD HH:mm:ss`
- 文章目录：`source/_posts/`
- 文件名：英文连字符格式，如 `my-new-post.md`
